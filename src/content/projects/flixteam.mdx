## üöÄ Solution

### üß± Architecture

- **Backend API:** Java + Micronaut, containerized and deployed on GCP.
- **Databases:** MongoDB for unstructured data, PostgreSQL for relational content like plans, invoices, users.
- **DevOps:** Used Vercel for frontends, Docker + Portainer for backend management.

### üñ• Frontend (Web)

- **Astro + React:** Built a clean UI with dynamic rendering for program listings, player integration, and pricing pages.
- **Stripe + Crypto Integration:** UI flows adapted for both payment types.
- **Admin Panel:** Custom internal tools for managing plans, users, stream links, and analytics.

### üì± Mobile App

- Built with **React Native** and deployed to iOS and Android.
- Features include:
- Login via Firebase
- Plan subscriptions and payments
- Live and VOD stream player
- Push notifications for new releases

### üí≥ Payments

- Stripe for card processing
- Integrated **crypto payments** via third-party wallet APIs
- Auto-renewal and plan expiry handling via cron + webhooks
